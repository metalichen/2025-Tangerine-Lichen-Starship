# Lichen Starship Project

## Captain Phylogeny Analysis
#### I first concatenated all assemblies into "ome.fa", and then ran Starfish Annotate on "ome.fa" to annotate captains.
```````````````````````````````````````````````````````````````````````````````
# Prepare assembly files
echo -e "lichens\tome.fa" > ome2assembly.txt
        
# Run starfish annotate on mtdb formatted genomes
starfish annotate --noCheck --separator _ \
--assembly ome2assembly.txt --tempdir temp --outdir lichens_temp \
--prefix lichens.tyr -p YRsuperfams.p1-512.hmm -P YRsuperfamRefs.faa --idtag tyr --targetFeat mRNA --threads 24
```````````````````````````````````````````````````````````````````````````````

#### I then added the captain sequences already annotated in the Xanthoria parietina and Xanthoria calcicola genomes to this larger lichen captain dataset.

``````````````````````````````````````````````````````````````````````````````````````````
cat lichens.tyr.filt.fas many_xanthorias_tyr.filt_intersect.fas > lichen_caps_plus_Xan.fasta
``````````````````````````````````````````````````````````````````````````````````````````

#### I then filtered the captains to remove sequences less than 203 amino acids in length

``````````````````````````````````````````````````````````````````````````````````````````
seqkit seq -m 203 lichen_caps_plus_Xan.fasta > filtered_lichen_caps.fasta
``````````````````````````````````````````````````````````````````````````````````````````

#### In order to remove sequences based on absence of active sites, I made a subset alignment file of representative reference captains from each family, and then used mafft --add to align the lichen caps to the alignment file to preserve positon of annotated active sites.

``````````````````````````````````````````````````````````````````````````````````````````
mafft --add filtered_lichen_caps.fasta --reorder sub_YR_reps.mafft > sub_reps_plus_filtered_lichen_caps.mafft
``````````````````````````````````````````````````````````````````````````````````````````

#### To find columns containing the active sites, this alignment file was cross-referenced with the annotated alignment file (supplementary file funTyr50_cap25_crp3_p1-512_activeFilt.clipkit) from Gluck-Thaler 2024. Using the Crypton sequences as a reference point, the columns corresponding with active sites 1, 2, 3, 4, 5, and 6 are 2065, 3330, 4597, 4627, 5124, and 5304, respectively.

#### When looking at this alignment, I observed that a large cluster of sequences that includes the Tangerine captains had a different site for active sites 3 and 4, which further suggests that these captains are structurally-distinct. These sequences were then moved into a new alignment file (tangerine_cluster_caps.mafft). Active sites 1, 2, 3, 4, 5, and 6 correspond with columns 2065, 3330, 4558, 4561, 5124, and 5304, respectively in this file. This 'tangerine' cluster was deleted from sub_reps_plus_filtered_lichen_caps.mafft and renamed sub_reps_plus_non_tangerine_cluster_caps.mafft.

#### I then removed sequences missing more than one residue at the respective annotated active site positions for the 'tangerine' and 'non-tangerine' alignment files to remove pseudogenes.

#### To get the representative sequences of the filtered active captains, I extracted the sequence IDs from the active captains alignment file, and filtered the size-filtered captain file to produce a file of only active captains.

``````````````````````````````````````````````````````````````````````````````````````````
seqkit seq -n sub_reps_plus_active_non_tangerine_cluster_caps.mafft > active_caps_ids.txt
seqkit seq -n active_tangerine_cluster_caps.mafft >> active_caps_ids.txt
seqkit grep -f active_caps_ids.txt filtered_lichen_caps.fasta > active_lichen_caps.fasta
``````````````````````````````````````````````````````````````````````````````````````````

#### I then ran clustering steps to produce active representative lichen captains.

``````````````````````````````````````````````````````````````````````````````````````````
mmseqs easy-cluster active_lichen_caps.fasta first_cluster tmp --min-seq-id 0.9 -c 0.8 --cov-mode 0 --alignment-mode 3 --cluster-reassign
rm -r tmp/
mmseqs easy-cluster first_cluster_rep_seq.fasta second_cluster tmp --min-seq-id 0.5 -c 0.8 --cov-mode 0 --alignment-mode 3 --cluster-reassign
mv second_cluster_rep_seq.fasta active_lichen_reps.fasta
``````````````````````````````````````````````````````````````````````````````````````````

#### I then added these representative lichen sequences to the full YR reference set and the outgroup crypton YRs to root with. I then aligned them using MAFFT E-INS-i.

``````````````````````````````````````````````````````````````````````````````````````````
cat active_lichen_reps.fasta YRsuperfamRefs.faa cryp3_cap25.faa > YR_reps_plus_lichen_reps.fasta
einsi YR_reps_plus_lichen_reps.fasta > YR_reps_plus_lichen_reps.mafft
``````````````````````````````````````````````````````````````````````````````````````````

#### I then used clipkit to remove columns with >90% gaps.

``````````````````````````````````````````````````````````````````````````````````````````
clipkit YR_reps_plus_lichen_reps.mafft -m kpic-gappy -g 0.9
``````````````````````````````````````````````````````````````````````````````````````````

#### Then the borders for the CB and CAT domains plus 50 flanking sites were found by cross-referencing the crypton sequences with the crypton sequences in the trimmed alignment file (supplementary file funTyr50_cap25_crp3_p1-512_activeFilt.clipkit) from Gluck-Thaler 2024. The columns outside of these borders were 1-18 and 489-863. These columns were manually trimmed. 

#### I then used IQ-Tree to construct a phylogenetic tree using this trimmed alignment file, and the tree was then rooted with the crypton sequences using phykit. 

``````````````````````````````````````````````````````````````````````````````````````````
iqtree3 -s YR_reps_plus_lichen_reps_trimmed.mafft.clipkit -B 1000 -m MFP --alrt 1000 -st AA -pre YR_reps_plus_lichen_reps.phylo
echo "CryptonF-1_RhMu" > root.txt
echo "CryptonF-2_PS_1p" >> root.txt
echo "CryptonF-5_PI_1p" >> root.txt
phykit root_tree YR_reps_plus_lichen_reps.phylo.treefile -r root.txt
``````````````````````````````````````````````````````````````````````````````````````````

## Captain HGT Phylogeny
#### To find captain clades with signatures of HGT between distantly related lichen symbionts, I referred to the clustering outputs to find every rep captain that was clustered with captains from other classes in R.
#### From this, found 12 representative captains that clustered with captain from other class during the second clustering step.

